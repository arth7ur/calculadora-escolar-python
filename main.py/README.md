# 📊 Calculadora de Média Escolar

Sistema desenvolvido em **Python** para facilitar o cálculo de médias acadêmicas e fornecer feedback automático sobre o status de aprovação do aluno.

## 🚀 Funcionalidades
- Entrada dinâmica do nome do aluno e notas bimestrais.
- Cálculo automático da média aritmética.
- Lógica de decisão para exibir o status:
  - **Aprovado**: Média >= 7.0
  - **Recuperação**: Média entre 5.0 e 6.9
  - **Reprovado**: Média < 5.0
- Formatação de saída com f-strings (Estilo Guanabara).

## 🛠️ Tecnologias e Conceitos
- **Linguagem:** Python 3
- **Conceitos:** Entrada/Saída de dados, Tipagem (float/str), Estruturas Condicionais (`if`, `elif`, `else`).
- **Versionamento:** Git e GitHub.
